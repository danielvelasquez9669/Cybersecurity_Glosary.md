# Cybersecurity_Glosary.md
Red Team: Offensive team trying to find vulnerabilities in a system. (Kali Linux and Metasploit)
Blue team: Defensive team that monitor, detect and answer attacks. (SIEM)
Pentesting: Controlled evualitation to find and exploit any vulnerabilities
Vulnerability: Weakness in a system that could get exploid by an attacker
Exploit: Code or Technic that take advantage of of a vulnerability
Malware: Malicious software design to damage, take or control systems.
Security Information and Event Management: Tools that centralize logs and detects anomalies
Open Source Intelligence: Recolection of public information for security ends
Privilege Escalation: Technic to obtain more privileges, but the ones you are supposed to have
DevSecOps: Security integration trought the whole cycle of development and software deployment
